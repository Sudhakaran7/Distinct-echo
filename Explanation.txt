Return the number of distinct non-empty substrings of text that can be written as the concatenation of some string with itself
(i.e. it can be written as a + a where a is some string).

Input Description:
String S as input.

Output Description:
Print the number of distinct non-empty substrings of text that can be written as the concatenation of some string with itself.

Sample Input:
abcabcabc

Sample Output:
3

Explanation:
The 3 substrings are "abcabc", "bcabca" and "cabcab".

Sample Input:
guviworld

Sample Output:
0

Sample Input:
guviivug

Sample Output:
1

Sample Input:
wewewe

Sample Output:
2

Sample Input:
qwerqwerqwert

Sample Output:
4

Sample Input:
tyhgtyhg

Sample Output:
1
